Proprietary Trading Firm Website Plan

ApexProp

"Apex" represents a peak performance in trading, while "Prop" clearly identifies the firm as a proprietary trading platform, making it easy to spot the site's purpose.

Purpose:

The ApexPro website lets traders register, complete evaluation phases, and access funded accounts. It provides a clear path to becoming a fully funded trader.

Scenarios:

  • How can I sign up for a funded account and start the evaluation process?
  • What are the rules and requirements I need to meet to pass the trading test and get funded?

Color Palette

Black (#000): Main Backgrounds, headers, text

Buff (#D4A373): Buttons, links, highlights

Jasmine (#F0C674): Hover states, highlights

White (#FFF): Page Background, spacing, text

Typography:

Montserrat: Headers, buttons, navigation

Roboto:Body text, content sections
